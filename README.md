# Defguard desktop client

Dekstop client for managing connection via instance of defguard. Built with tauri and React.js

## Development

### Install pnpm
```bash
npm i -g pnpm
```
### Install dependencies
```bash
pnpm install
```
### Start dev server
```bash
pnpm dev
```

## Building
### After installing deps
```bash
pnpm build
```
